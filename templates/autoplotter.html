AutoPlotter

AutoPlotter makes various plots from real CMS3 data and Monte Carlo simulations. It exports these plots as .pdf files which are then displayed on an easily accessible webpage. The example page is hosted by UCSD. All of the plot-related code is written in C++, with some Python and JavaScript responsible for the functionality of the web interface.
