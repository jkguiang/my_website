Collapsable Links




The JavaScript for this project was mainly written by Nick Amin, a graduate student working for the Campagnari group. It is responsible for taking a .json produced by the data parser, and all of the links and pasting all of the associated pictures below, as well as allowing for collapsable links and several other features not displayed here.

Graphs




The graphs shown here were all generated by the plotter program linked on the previous page. They are simple, two dimensional histograms from the Matplotlib library. The x and y-axis labels were retrieved directly from the dstat logfiles. Here, epoch refers to the time, where all graphs are normalized to a time length of one (for greater readability). The y-axis labels refer to various quantities describing CPU, memory, and network usage.

Summaries




This summary information is printed like this for all jobs evaluated by the log parser. It describes the jobs that were unsuccessful, as well as how much information is missing in total. It is displayed as the raw .json output because it is readable enough, such that it was not worth the time and effort to format further.
