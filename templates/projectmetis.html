Metis: Log Module

Metis is a program designed to take data from the CMS detector at the LHC and grouping particular samples into smaller, more usable .root files for further research. I was responsible for designing and implementing a module that took metadata concerning this process and produced plots that allow the user to better understand how the program as a whole is performing.
