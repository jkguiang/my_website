Metis: Log Module

Metis is a program designed to take data from the CMS detector at the LHC and grouping particular samples into smaller, more usable .root files for further research. I was responsible for designing and implementing a module that took metadata concerning this process and produced plots that allow the user to better understand how the program is performing.

Source Code

Plotter

I made each different plot-type into a function, allowing for the simple addition of new plots if necessary. I decided to use Matplotlib because it's easy to use and the plots we needed were not particularly complicated.

JSON Maker

The web interface, written by a graduate student working for the Campagnari group, populates itself from a .json. This program generates a .json containing plot paths and other information.

Log Parser

This program parses dstat log files generated during Metis's copying process. It essentially takes all of the dstat information, such as CPU or memory usage, and organizes it into a dictionary.
