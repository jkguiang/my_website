AutoDQM

AutoDQM is a statistical tool for Data Quality Management (DQM) at the Large Hadron Particle Collider. Before AutoDQM, DQM shifters were tasked with looking at hundreds of DQM histograms, looking for hard-to-spot issues in data collection. My tool runs long established statistical tests on these graphs and outputs outliers on a simple, but effective, GUI that expedites the shifters' task.

Highlights

Web Interface: PHP

AutoDQM's GUI is writting in PHP, mainly to allow it to dynamically source output histograms. More speifically, PHP is used to retrieve the output of a user's specific query, which is tagged by a unique timestamp calculated on query submission.

Github »

Web Interface: Javascript

The backbone of AutoDQM's functionality is written in Javascript, with heavy use of JQuery. Javascript-based features include dynamic mulit-page lists, search bars, unique URLs generated for each query submission, and many more.

Github »

Statistical Arsenal

AutoDQM calculates the χ2 between a data and reference histogram. It also uses the Kolmogorov-Smirnov test, and it plots output histograms with either cell-by-cell pull values for 2D plots or an overlay of the reference and data histograms for 1D plots.
